# vihn-portfolio

Personal portfolio of **Nguyễn Thành Vinh** — Software Engineering student at VKU.

## Tech Stack

- **Framework:** Next.js 16 (App Router)
- **Styling:** Tailwind CSS v4, shadcn/ui
- **Language:** TypeScript

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Structure

```
app/          # Routes & layouts
components/   # UI components & page sections
lib/          # Data & utilities
public/       # Static assets & CV PDF
```

## Routes

| Route | Description |
|-------|-------------|
| `/` | Main portfolio (About, Projects, Resume, Contact) |
| `/cv` | Redirects to CV PDF |
